# Endpoint Security Standard
Lists required security controls for laptops, desktops, and mobile devices (e.g., antivirus, updates, firewalls).

## Purpose
To define mandatory security requirements for all endpoints, including laptops, desktops, mobile devices, and servers.

## Scope
All company-owned or Bring your Own Device (BYOD) devices accessing organisational systems or data.

## Requirements
- All endpoints must run approved antivirus/EDR solutions.
- Operating systems must be supported and kept up to date.
- Full disk encryption (e.g., BitLocker, FileVault) is required.
- Screens must auto-lock after 10 minutes of inactivity.
- USB storage devices must be restricted or monitored.
- Admin privileges must be granted only when necessary.
- Devices must connect only to secure, trusted networks.
- Firewall must be enabled on all endpoints.

## Monitoring
Security tools must report device health, patch status, and threats to central management.

## Enforcement
Non-compliant devices may be blocked from accessing company systems.
